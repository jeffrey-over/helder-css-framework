Getting started

 Download the minified Helder stylesheet and use this basic HTML template below. That's it! You're now ready to start your amazing project. Use the links in the menu or scroll down to see the documentation of the components Helder contains. 
<!doctype html>
<html>
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">

    <link rel="stylesheet" href="css/main.css">
    <link rel="stylesheet" href="css/custom.css">    
    <title>Your amazing project</title>
</head>
<body>
    
</body>
</html>
